Have you ever been stuck searching the web trying to figure out grid properties? Well look no further, This is a handy cheatsheet to help you understand how grid properties work.

Grid Template Rows

Specifies the size(s) of the rows. The values listed are separated by spaces and each value defines specific height values of the corresponding row.

  
.grid-container {
display: grid;
grid-template-rows: 100px 300px;
}

Grid Template Columns

Specifies the size(s) of the columns. The values listed are separated by spaces and each value defines specific height values of the corresponding column.

  
.grid-container {
display: grid;
grid-template-columns: auto auto auto auto;
}

Grid Template Areas

Specifies the grid layout using named items. The grid-area property can be used to name grid items and then reference that name in the grid-template-areas property. Using a period sign will refer to a grid item with no name.

  
.item1 {
grid-area: myArea;
}
.grid-container {
display: grid;
grid-template-areas: "myArea myArea . . .";
}

Grid Auto Rows

Property sets a size for the rows in a grid container. This property will only affect the rows where the size is not set.

  
.grid-container {
display: grid;
grid-auto-rows: 150px;
}

Grid Auto Columns

Property sets a size for the columns in a grid container. This property will only affect the rows where the size is not set.

   
.grid-container {
display: grid;
grid-auto-columns: 50px;
}

Grid Auto Flow

Property controls how auto-placed items get inserted in the grid.

    
.grid-container {
display: grid;
grid-auto-flow: column;
}
